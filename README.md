# Supermarket Sales Analysis (SQL Project)

## Overview
This project performs data analysis on a supermarket sales dataset using SQL to extract meaningful business insights.

## Project Structure
- data/ : Contains raw dataset
- sql/ : Contains SQL queries
- report/ : Contains project report

## Objectives
- Analyze sales by branch and city
- Identify top product lines
- Compare customer types
- Analyze payment methods

## Tools
- MySQL
- MySQL Workbench
- Excel
- GitHub
  ## Dataset
Publicly available supermarket sales dataset used for academic and learning purposes.
